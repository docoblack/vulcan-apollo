Security & Encryption

Multiple layers of protection.

Encryption at Rest

AES-256 encryption on all agent memory, vector databases, documents, and logs. Full disk encryption (LUKS / BitLocker) on all servers.

Encryption in Transit

All communication uses TLS 1.3 with perfect forward secrecy. Certificate pinning available for managed deployments.

Application-Level Encryption

Sensitive fields (PII, financial data, credentials) are encrypted even before they reach the database. Strict tenant isolation and zero-trust architecture.

Operational Security

Agents run in hardened Docker containers with minimal privileges. Prompt injection protection, output validation, immutable encrypted audit logs, and automated vulnerability scanning.

Human Oversight & Access Control

Critical actions require human approval. Least-privilege access — our team has no default access to your agent or data. Optional Customer-Managed Keys (BYOK).

FAQ

Can I run the agent completely offline?

Yes. We fully support air-gapped deployments using local models.

Who can access my data?

Only you and authorized members of your team. We have zero access unless you explicitly grant temporary support access.

What if I stop using the service?

You receive a full encrypted export of your agent's memory and configurations. All data is permanently deleted from our systems.
